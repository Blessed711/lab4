# Lab 4: Алгоритми на графах

## Опис

У даній лабораторній роботі реалізовано декілька алгоритмів та структур даних для роботи з графами.

## Зміст

- **area_frame.py** — побудова каркаса площі графа.
- **dsu.py** — реалізація структури неперетинних множин (Disjoint Set Union).
- **min_time.py** — знаходження мінімального часу доставки або досягнення вершини.
- **mod_dijkstra.py** — модифікований алгоритм Дейкстри для графів з додатковими умовами.

## Інструкції

1. Запусти потрібний скрипт для відповідного завдання.
2. Переконайся, що всі вхідні дані подані у відповідному форматі.
3. Перевір результати згідно з умовою задачі.
